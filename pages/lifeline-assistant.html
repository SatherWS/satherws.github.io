Assurance Wireless Web App Re-development


WATCH DEMO VIDEO

OVERVIEW

My team and I built this site for a school project. We were tasked with creating a software application for a business, we chose the company Assurance Wireless. The problem Assurance Wireless was having was that they kept getting lifeline applications from addresses that did not exist. This wasted a lot of time because employees of the company needed to manually check the addresses of online applications for their free services. Also one of the technical leads of the company needed a better interface for processing applications. As a bonus feature we decided to attempt to implement a customer support chatroom.


FEATURES

GPS API Integration via GeoPy

This was the most important feature we've implemented. This feature takes the address that was inputted by the user and checks if it has valid gps coordinates. If the address does not produce valid GPS coordinates the user will receive an error and not be able to submit their lifeline application.
